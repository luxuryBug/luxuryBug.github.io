1、`:checked`选择器只能选择到表单元素，不能选择到div
2、`:enable`选择器选择启用的表单元素,`:disabled`选择器选择禁用的表单元素
3、`:default`选择器选择默认选中的表单元素值，只对checkbox、radio生效

4、`:indeterminate`选择器用于选择处于不确定状态的表单元素，如：checkbox、radio、progress
5、`:in-range`选择器用于选择值在指定范围内的元素，`:out-of-range`选择器选择值在指定范围之外的所有元素,不建议使用，IE edge上的表现是错误的
6、`:invalid`选择器用于选择值不合法的，IE10以及IE10以上才支持
